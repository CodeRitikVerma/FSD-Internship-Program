The tools
of the trade.

Languages and frameworks get the spotlight, but the tools below are what you'll actually open every single day. Treat them as muscle memory, not afterthoughts.

— 01
Git
Version control
Track every change, branch fearlessly, and undo mistakes without panic. Learn init, add, commit, push, pull, branch, merge, and rebase.
— 02
GitHub
Code hosting
Where your code lives publicly. Repos, README files, pull requests, issues, and GitHub Pages — your portfolio's permanent home.
— 03
Postman
API testing
Hit every endpoint before your frontend ever does. Build collections, save environments, attach JWT tokens, debug responses.
— 04
VS Code
Code editor
Your daily workspace. Configure extensions, integrated terminal, Git integration, Prettier, ESLint, and live server.
— 05
DevTools
Browser debugging
Chrome's Elements, Console, Network, and Application tabs. Inspect requests, debug JavaScript, profile performance, simulate devices.
— 06
npm
Package manager
Install, update, and run JavaScript packages. Understand package.json, dependencies, scripts, and the node_modules folder.
— 07
MySQL
Database
A relational database. Write SQL queries, design schemas with Workbench, and connect from Spring Boot via JDBC.
— 08
IntelliJ
Java IDE
The serious Java IDE. Run Spring Boot apps, debug with breakpoints, refactor confidently, and use the database tool window.
— 09
Maven
Build tool
Manage Java dependencies via pom.xml. Understand the build lifecycle — compile, test, package, install — and Spring starters.
— 10
Vercel
Deployment
One-click frontend deploys. Pair with Render or Railway for backend hosting. Push to main, see it live in seconds.
Week 01
Days 01 — 07 · Frontend foundations

The building blocks every web page is made of.

Markup, styling, layout, and a component library — by the end of the week you'll have a deployed portfolio site as proof.

Day01
Setup, Git basics & HTML fundamentals
Install VS Code, Git, and a browser. Configure Git with your name and email. Learn the anatomy of an HTML page — DOCTYPE, head, body, semantic tags, headings, paragraphs, lists, links, and images. Run your first git init and git commit.
→
Day02
HTML forms, tables & embedded media
Master input types, labels, select, textareas, validation attributes, table semantics, video/audio, and iframes. Build a registration form and a data table.
→
Day03
CSS basics — selectors, box model, color
Inline / internal / external CSS, selectors (class, id, attribute, pseudo), specificity, the box model, units (px, rem, em, %), and color systems.
→
Day04
Flexbox & CSS Grid layouts
Flexbox for one-dimensional flow, CSS Grid for two-dimensional structure. Build a fully responsive landing page from a blank file.
→
Day05
Responsive design, media queries & DevTools
Mobile-first approach, viewport meta tag, breakpoints, fluid typography with clamp(), CSS transitions and keyframe animations. Master Chrome DevTools — Elements, device toolbar, Network tab.
→
Day06
Bootstrap 5 — grid system & components
Bootstrap CDN setup, the 12-column grid, containers, rows, breakpoints, and core components: buttons, cards, navbars, modals, forms, badges.
→
Day07
GitHub, deployment & portfolio site
Combine HTML, CSS, and Bootstrap into a multi-page portfolio. Create a GitHub account, push your code with branches and pull requests, write a clean README, and deploy live via GitHub Pages or Netlify.
→
Week 02
Days 08 — 15 · JavaScript & React

From static pages to interactive applications.

Vanilla JavaScript first, then a leap into component-based UI with React. By day 15, your app fetches real data from a real API.

Day08
JavaScript core — variables, types, operators
let / const / var, primitive vs reference types, template literals, type coercion, comparison operators, conditionals, and loops.
→
Day09
Functions, arrays & objects
Function declarations vs expressions, arrow functions, rest/spread, array methods (map, filter, reduce), object literals, and destructuring.
→
Day10
DOM manipulation & events
querySelector, modifying styles and attributes, creating and removing elements, event listeners, event bubbling, and form handling.
→
Day11
Async JavaScript — promises, async/await, fetch
The event loop, callbacks, promises, async/await, error handling with try/catch, and the Fetch API against a real public endpoint.
→
Day12
Node.js, npm, Vite & React essentials
Install Node.js and npm, understand package.json and node_modules, spin up a Vite project, master JSX syntax, build functional components, pass props, and grasp the virtual DOM.
→
Day13
State, hooks & event handling
useState for local state, useEffect for side effects, controlled forms, conditional rendering, list rendering with keys, and lifting state up.
→
Day14
React Router & API integration
react-router-dom, routes and nested routes, Link and useNavigate, fetching with Axios in useEffect, and handling loading/error states.
→
Day15
Mini project — React todo or weather app
Apply hooks, components, and API calls in a real app. Style with Bootstrap or Tailwind, then deploy on Vercel or Netlify.
→
Week 03
Days 16 — 22 · Backend & databases

Java, Spring Boot, and everything behind the API.

The heaviest week. Layered architecture, JPA, validation, exception handling, and JWT-secured endpoints — production patterns you'll see on day one of any backend job.

Day16
Java refresher & OOP fundamentals
Java syntax, classes and objects, constructors, inheritance, polymorphism, abstraction, encapsulation, interfaces, collections, and exceptions.
→
Day17
Spring Boot setup, Maven & IntelliJ / VS Code
Install JDK 17+, generate a project on start.spring.io, understand Maven (pom.xml, dependencies, lifecycle), the @SpringBootApplication entry point, and the layered architecture pattern.
→
Day18
REST APIs with Spring Web & Postman
@RestController, @GetMapping / @PostMapping / @PutMapping / @DeleteMapping, @PathVariable, @RequestBody, ResponseEntity. Install Postman, build collections, and test every endpoint thoroughly.
→
Day19
MySQL, Workbench & Spring Data JPA
Install MySQL Server and MySQL Workbench, write your first SQL (CREATE, INSERT, SELECT, JOIN), connect Spring Boot via JDBC, define @Entity classes, and run real CRUD via JpaRepository.
→
Day20
Service layer, DTOs & validation
Move business logic into @Service classes, decouple API contracts with DTOs, and validate input with @Valid and Bean Validation annotations.
→
Day21
Exception handling & relationships
Global @ControllerAdvice handlers, custom exceptions, and JPA relationships — @OneToMany, @ManyToOne, @ManyToMany with cascade and fetch types.
→
Day22
Spring Security & JWT authentication
Configure Spring Security, BCrypt password encoding, login/register endpoints, JWT generation and validation, and role-based access control.
→
Week 04
Days 23 — 30 · Full stack integration

The capstone. Where everything clicks together.

Design, build, integrate, deploy, document. Walk away from day 30 with a live full-stack project on your GitHub — worth more than any certificate.

Day23
Project planning, schema & Git workflow
Pick a capstone (task manager, blog, e-commerce). Design the MySQL schema with Workbench, define endpoints, sketch wireframes, set up two GitHub repos with proper README, .gitignore, and a feature-branch workflow.
→
Day24
Backend build — entities & APIs
Create JPA entities, repositories, services, and REST controllers. Implement full CRUD and verify every endpoint with Postman before touching the UI.
→
Day25
Backend auth & CORS configuration
Wire up JWT-based login and registration, secure protected routes, and configure CORS so the React dev server can talk to Spring Boot.
→
Day26
Frontend build — pages & routing
Login, register, dashboard, list view, detail view — all with React Router. Style with Bootstrap and ensure mobile responsiveness.
→
Day27
Connect frontend to backend with Axios
Axios instance with base URL, JWT interceptors, token storage, protected routes, and real API calls wired to every CRUD page.
→
Day28
Testing, debugging & polish
End-to-end manual testing, loading and error UI states, form validation messages, basic JUnit tests, and serious DevTools work.
→
Day29
Deployment — frontend & backend
React build to Vercel or Netlify, Spring Boot JAR to Render or Railway, a free-tier database (Aiven, Railway), and environment variables wired up.
→
Day30
Documentation, demo & resume polish
A clear README, screenshots, the live link. A short demo video. GitHub and LinkedIn updated. Resume ready. Interviews next.
→

Spend the first hours learning, the next hours coding, and the last hour pushing to GitHub. Consistency beats intensity — every single time.

— Ritik Verma
